Twinkle



ABOUT

To create this gif, I manipulated a photo's code to cause a glitch in the file -- resulting in color changes and breaking in the photo. I decided to use a picture of a baby doll because I thought the pureness and beauty of the doll would look interesting with the effect of a glitch, which can cause a scary a eery effect. It shows how beauty can be easily manipulated.
